-PROPOSTA DO PROJETO
Desenvolver um sistema web para correção de provas, com o objetivo de automatizar o processo de avaliação, estudo para avaliações e atividades,creduzir erros e otimizar o tempo dos professores e dos alunos também. 

O SISTEMA PERMITIRÁ:
- Cadastro de provas
- Cadastro de questões
- Correção automática e manual de questões
- Sistema de pontuação
- Armazenamento de provas corrigidas
- Consulta de provas anteriores dos alunos


ATIVIDADES DO PROJETO
- Levantamento de requisitos (Concluido)
- Planejamento do sistema
- Modelagem (diagramas) (Concluido)
- Desenvolvimento do sistema
- Implementação da lógica de correção
- Testes
- Documentação final

-FUNCIONALIDADES DO SISTEMA:
- Cadastro de professores
- Cadastro de alunos
- Criação de provas
- Cadastro de questões (objetivas e dissertativas)
- Correção automática (questões objetivas)
- Correção manual (questões dissertativas)
- Cálculo de notas
- Histórico de provas dos alunos
